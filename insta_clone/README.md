# Instagram Clone

리액트로 만든 인스타그램 클론 앱입니다.

## 기능

- 인스타그램 스타일의 헤더
- 스토리 섹션
- 피드 포스트 (좋아요, 댓글, 저장 기능)
- 반응형 디자인
- 사이드바 (추천 친구, 프로필 정보)

## 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용된 이미지

- 피드 이미지: `images/img01.png ~ img10.jpg`
- 프로필 이미지: `man.png`, `woman.png`

## 기술 스택

- React 18
- Vite
- CSS3
